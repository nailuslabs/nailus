  ## 🔧 Installation

| Intégration | code | Lien |
|-------------|------|------|
| **Vite** | pnpm add -D @nailus | [Guide Vite](https://nailus.dev/vite) |
| **Nuxt** | pnpm add -D nailus @nailus/nuxt | [Guide Nuxt](https://nailus.dev/nuxt) |
| **Astro** | pnpm add -D nailus @nailus/astro | [Guide Astro](https://nailus.dev/astro) |
| **Webpack** | pnpm add -D nailus @nailus/webpack | [Guide Webpack](https://nailus.dev/webpack) |
| **CDN Runtime** | <script src="https://cdn.jsdelivr.net/npm/@nailus/runtime"></script> | [Guide CDN](https://nailus.dev/cdn) |
| **CLI** | pnpm add -D nailus | [Guide CLI](https://nailus.dev/cli) |
| **VS Code** | go to marketPlace | [Extension](https://marketplace.visualstudio.com/items?itemName=nailus.vscode) |
| **ESLint** | pnpm add -D @nailuscss/eslint-config | [Configuration](https://nailus.dev/eslint) |
| **PostCSS** | pnpm add -D nailus @nailus/postcss | [Guide PostCSS](https://nailus.dev/postcss) |
